About . . .

This is a Python RESTful Web Application of different stores and their corresponding items developed using Flask and Flask_RESTful.


The API conforms to REST standards supporting GET / PUT / POST / DELETE requests.


Store and Item data is stored in an SQLite Database.


User registration / authentication / login implemented using JSON Web Tokens and Flask_JWT.
